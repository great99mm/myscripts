# My Scripts

个人脚本仓库

## 脚本列表

### qbmanager - qBittorrent 自动管理

安装：
```bash
git clone git@github.com:great99mm/myscripts.git
cd myscripts/qbmanager
sudo ./install.sh
```

使用：
```bash
qbmanager          # 进入管理菜单
qbmanager start    # 启动
qbmanager stop     # 停止
qbmanager status   # 查看状态
qbmanager update   # 更新脚本
```

详见 [qbmanager/README.md](qbmanager/README.md)

### mteam2qb - M-Team 收藏导入

安装：
```bash
git clone git@github.com:great99mm/myscripts.git
cd myscripts/mteam2qb
sudo ./install.sh
```

使用：
```bash
mteam2qb          # 全部导入
mteam2qb 2        # 从第2页开始
mteam2qb 2-5      # 只跑第2-5页
```
